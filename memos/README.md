# Starweaver Memos

This directory holds working notes, design comparisons, implementation evidence, roadmap decisions, and release-preparation reminders.

Use `memos/` for material that helps active design and implementation work. Use `spec/` for Starweaver's long-lived architecture baseline. Curate or retire memo content before stable public releases.

Current memos:

- `implementation-todo.md` — current implementation roadmap, landed milestone evidence, next application-runtime milestones, validation gates, and review checklist.
- `agent-sdk-foundation-plan.md` — merged Agent SDK P0/P1 foundation record with landed evidence, public API decisions, focused tests, and remaining SDK-deepening items.
- `builtin-tool-alignment-audit.md` — native/provider tool alignment, first-party tool bundle names, coverage evidence, and remaining replay fixture gaps.
- `sdk-host-tool-gap-report.md` — host-backed SDK tool gap report for search, scrape, media URL understanding, downloads, and document-conversion skill workflow planning.
- `pre-1.0-reference-notes.md` — reference mapping, landed snapshot, remaining pre-1.0 gaps, and public-positioning cleanup reminders.
