# Starweaver Memos

This directory holds working notes, design comparisons, implementation evidence, and release-preparation reminders.

Use `memos/` for material that helps design work today and can be curated before a stable public release. Use `spec/` for Starweaver's long-lived architecture baseline.

Current memos:

- `implementation-todo.md` — detailed implementation roadmap, replay TODO matrix, Pydantic AI feature coverage, ya-agent-sdk integration plan, validation gates, and review checklist.
- `pre-1.0-reference-notes.md` — reference notes and release preparation reminders.
