# Starweaver Python

`starweaver` is the planned in-process Python SDK and binding package for
Starweaver.

The current package is an initial native-extension scaffold. It establishes the
package layout, versioning, linting, CI, wheel build, and PyPI release path
before the agent, tool, session, and streaming APIs are filled in.

Supported Python versions are CPython 3.11 through 3.13. Local development and
single-version CI jobs default to Python 3.13.
